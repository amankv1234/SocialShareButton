## üîê Reporting a Vulnerability

If you discover a security vulnerability in SocialShareButton, please report it responsibly.

We request that you do NOT create a public GitHub issue for security vulnerabilities.

Instead, please report the issue privately by:

- Using GitHub's **[Private Vulnerability Reporting](https://github.com/AOSSIE-Org/SocialShareButton/security/advisories/new)** ‚Äî
 click "Report a vulnerability" under the [Security](https://github.com/AOSSIE-Org/SocialShareButton/security) tab of this repository.
- For non-sensitive follow-up, you may also reach us on the
    [Discord server](https://discord.gg/hjUhu33uAn).
Please include the following information in your report:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

---

## üì¶ Supported Versions

As this is a lightweight project without versioned releases, security fixes will be applied to the `main` branch.

We appreciate responsible disclosure and will:
